# AutoHub | 3D Supercar Showroom & Data Portal
An interactive web application built with **Flask** and **SQLite** for managing premium vehicle inventory.

### Key Features
* **3D Visualization**: Real-time 3D model rendering for Lamborghinis and Ferraris.
* **Data Analytics**: Live dashboard calculating total inventory volume and value.
* **CRUD Operations**: Full ability to Create, Read, Update, and Delete vehicle records.
* **Business Export**: Integrated **Pandas** logic to export inventory reports to Excel.

### Tech Stack
* **Backend**: Python (Flask), SQLite, Pandas
* **Frontend**: HTML5, Bootstrap 5, Three.js (via Google Model-Viewer)
* **Animation**: CSS3 Parallax Racing Background
